# Personal Arcane Template Registry

This repo is set up as a personal template registry for [Arcane](https://getarcane.app), a Docker management tool.

To use these templates in Arcane:

1. Open Arcane Customization
2. Go to Templates
3. Go to Manager Registries
4. Add Registry
5. Enter the registry URL: `https://raw.githubusercontent.com/JourneyOver/selfhosted_templates/main/registry.json`

## Note for Visitors

- **Feel free to use or adapt** any of the stacks for your own needs.
- **No requests or issues** will be addressed, as this repository is maintained solely for personal use.
